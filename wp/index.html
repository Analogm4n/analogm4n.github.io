Timelapse

December 22, 2024

In this machine we will access to an SMB share that contains a zip file. Cracking it, we extract an encrypted PFX file. From the PFX file an SSL certificate and a private key can be extracted. We discover a login credentials for an user. The user can read the local passwords for machines in the domain.

Tabby

December 21, 2024

Tabby is an easy box. Enumeration of websites reveals that one is vulnerable to LFI. With this we can read credentials, login and upload a webshell. Inside the system we found a zip file, once cracked, we use the creds to login as ash user. This user can create a privileged container that allow us to modify the box as root.

HackTheBox | Linux | Easy | LFI | groups

Headless

December 17, 2024

Headless is an Linux machine that features a website that is vulnerable to blind Cross-Site Scripting. This vulnerability is leveraged to steal an admin cookie and gain access as admin. Beside, the page is vulnerable to command injection. Inside the machine, there is an script that does not use absolute paths, which is leveraged to get a shell as root.

HackTheBox | Linux | Easy | RCE | XSS | sudo

ServMon

December 14, 2024

ServMon is an easy Windows machine featuring an HTTP server that hosts an NVMS-1000 instance. This is found to be vulnerable to LFI. We can SSH to the server as a second user. The app NSClient++ contains functionality to create scripts that can be executed in the context of `nt authority\system`.

HackTheBox | Windows | Easy | RCE | Directory Traversal

Photobomb

Photobomb is a Linux machine where plaintext credentials are used to access a web application with a "Download" functionality that is vulnerable to blind command injection. Once access to the machine is gained, a misconfigured shell script is leveraged to gain elevated privileges as it can be executed with "sudo".

HackTheBox | Linux | Easy | RCE | sudo
