Netmon

November 24, 2024

Netmon is an easy difficulty Windows box with simple enumeration and exploitation. PRTG is running, and an FTP server with anonymous access allows reading of PRTG Network Monitor configuration files. The version of PRTG is vulnerable to RCE which can be exploited to gain a SYSTEM shell.

Lame

November 07, 2024

In this first WriteUp, we will explore the Lame machine from Hack The Box. Lame was the first box released and is characterized by its low difficulty, ideal for those who are just starting out in the world of pentesting like me. In this analysis, we will demonstrate how to gain root access using a single exploit, taking advantage of a specific vulnerability in the box.
